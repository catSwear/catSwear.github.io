如何处理npm与nodejs版本冲突

在开发过程中npm版本的更新和定向更新可能导致npm和nodejs版本冲突，后续安装包的时候报错

如果使用

$ npm install npm

也会提示更新版本就需要别的方法

方法：

卸载环境变量和所有npm的信息

然后卸载nodejs 去官网下载最新版重新安装

一、环境变量：

1.用户环境变量和系统变量都要删除

二、npm残留文件

使用一段时间会出现npm和npm_cache两个文件夹。AppData是隐藏文件夹要显示出来，删掉之后。再卸载nodejs就卸载干净了
